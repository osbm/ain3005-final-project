Library Automation System

Note: This is a demo application. You can do whatever you want with it. All the data will reset after you restart the docker compose.

You can do following things without logging in:

  • Search the library for books and users
  • Donate a book to our collection

After logging in you can also:

  • Borrow books
  • Return books
  • Reserve books
  • Cancel reservations
  • Buy books
  • Donate books
  • See your own profile
  • See your own history
  • Delete your account

After logging in as an admin you can also:

  • See all other user statistics
  • Delete/edit users books fines reservations
  • See all books
